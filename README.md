# ai2god.club

Source code and markdown documents of [https://ai2god.club](https://ai2god.club).

This website is generated by [GitSite](https://gitsite.org) and served as static website.


## Download

You can download all markdown documents and generated static web site from [Release](https://github.com/Gangge1991/ai2god/releases).

## Deploy Guide
npm install -g gitsite-cli
npm update -g gitsite-cli
gitsite-cli build -o dist -v

### 启动nginx代理
mv traffic ../
cd ../traffic
docker compose up -d

### 启动web
cd ai2god
docker compose up -d
